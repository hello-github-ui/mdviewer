# 虚拟环境

> 安装：`pip install virtualenv`，
>
> 参考文档：

* 创建一个局部的隔离的虚拟环境

> 语法：`virtualenv 虚拟环境名称`
>
> 例：`virtualenv ENV`
>
> 如果想在创建虚拟环境时指定Python的版本，使用类似下面格式的命令即可：
>
> `virtualenv -p C:/Python/27/python.exe ENV`

* 激活虚拟环境

> 进入 `ENV` 目录的 脚本目录（`Scripts`） 后，执行 `active.bat` 或 `active`

* 在激活状态下，想干啥干啥
* 退出虚拟环境
* 删除虚拟环境

* PyCharm 中使用虚拟环境
* 虚拟环境进阶补充
