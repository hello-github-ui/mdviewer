```python
# 基本写法
# [表达式 for 变量 in 列表]
# [表达式 for 变量 in 列表 if 条件]
```


```python
li = [1, 2, 3, 4, 5, 6, 7, 8, 9]
print([i*2 for i in li]) # 元素的值为表达式的值(i*2)，i 的取值来自于li列表
```

    [2, 4, 6, 8, 10, 12, 14, 16, 18]
    


```python
# for 循环方式
li = [1, 2, 3, 4, 5, 6, 7, 8, 9]
li2 = [] # 声明一个空列表
for i in li:
    li2.append(i*2) # 使用 append 追加元素
print(li2)
```

    [2, 4, 6, 8, 10, 12, 14, 16, 18]
