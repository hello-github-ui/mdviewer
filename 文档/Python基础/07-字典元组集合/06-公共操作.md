```python
# 一、+ 拼接，适用于 字符串、列表、元组
# 1.字符串
str1 = 'a'
str2 = 'b'
result = str1 + str2
print(result)
```

    ab
    


```python
# 2.列表
list1 = [1, 2]
list2 = [3, 4]
result = list1 + list2
print(result)
```

    [1, 2, 3, 4]
    


```python
# 3.元组
t1 = (1,)
t2 = (0.3, '2')
result = t1 + t2
print(result)
```

    (1, 0.3, '2')

# 4.集合
